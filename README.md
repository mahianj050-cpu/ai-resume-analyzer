# AI Resume Analyzer

An AI-powered Resume Analyzer that intelligently ranks resumes based on a given job description using Natural Language Processing (NLP) and Semantic Similarity.

## Features

- User Login & Registration
- Upload Multiple Resumes
- Job Description Analysis
- Semantic Resume Matching using Sentence Transformers
- Skill Extraction using spaCy
- Resume Ranking based on Similarity Score
- Match Score Visualization using Charts
- Download Matched Resumes as ZIP
- User Feedback & Sentiment Analysis
- Admin Dashboard for Feedback Management

## Tech Stack

- Python
- Flask
- Sentence Transformers (SBERT)
- spaCy
- Scikit-learn
- PDFPlumber
- Docx2txt
- SQLite

- CSS

## Installation

Clone the repository

```bash
git clone https://github.com/mahianj050-cpu/ai-resume-analyzer.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
